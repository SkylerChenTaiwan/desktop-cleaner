# 系統總覽

> 最後更新：2024-12-10

## 專案簡介

**專案名稱**：Desktop Cleaner
**專案描述**：macOS 自動清理工具，每天定時將「下載」資料夾和「桌面」上超過三天的檔案移到垃圾桶。

## 技術棧

| 層級 | 技術 |
|------|------|
| 程式語言 | Swift |
| 排程機制 | macOS LaunchAgent |
| 目標平台 | macOS |

## 專案結構

```
desktop-cleaner/
├── Sources/                # Swift 原始碼
│   └── DesktopCleaner/
├── LaunchAgent/            # LaunchAgent plist 設定
├── STATE/                  # 專案狀態文件
└── .snapshots/             # 程式碼結構快照
```

## 核心功能

| 功能 | 說明 |
|------|------|
| 定時執行 | 每天 22:00-23:00 自動執行 |
| 掃描資料夾 | 檢查「下載」和「桌面」資料夾 |
| 時間判斷 | 找出超過 72 小時（三天）的檔案 |
| 安全刪除 | 將檔案移到垃圾桶（可救回） |

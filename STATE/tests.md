## 執行測試

```bash
# 執行所有單元測試
npm test

# 執行特定測試檔案
npm test -- --testPathPattern="XxxService"

# 執行 E2E 測試
npx playwright test

# 執行特定 E2E 測試
npx playwright test xxx.spec.ts
```

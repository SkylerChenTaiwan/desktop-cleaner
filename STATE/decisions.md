# 技術決策記錄

> 記錄專案中的重要技術決策

---

## 決策模板

使用 ADR (Architecture Decision Records) 格式：

```markdown
## ADR-XXX: [決策標題]

**日期**：YYYY-MM-DD
**狀態**：提議中 / 已接受 / 已廢棄 / 已取代
**決策者**：[誰做的決定]

### 背景
為什麼需要做這個決定？

### 決策
我們決定...

### 選項考量

| 選項 | 優點 | 缺點 |
|------|------|------|
| 選項 A | ... | ... |
| 選項 B | ... | ... |

### 後果
這個決定會帶來什麼影響？

### 參考資料
- [連結]
```
